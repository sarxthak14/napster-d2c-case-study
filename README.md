# Napster Clothing — D2C E-Commerce Brand Launch
**Project Overview:** A complete end-to-end launch of a Direct-to-Consumer (D2C) fashion brand, spanning e-commerce website architecture, search engine optimization (SEO), social media strategy, and email marketing frameworks.

> 🌐 **Note on Live Site:** The domain `napsterclothing.in` is currently offline due to hosting expiration. Full visual archives, strategy files, and design mockups are documented below to preserve the technical architecture of the launch.

## 🚀 Key Achievements & Core Responsibilities

### 1. Web Design & UI/UX Architecture
* Built and structured the entire e-commerce store layout, handling product catalog mapping, collections, and checkout navigation flows.
* Created custom visual banners, product page layouts, and UI assets using **Canva** to ensure a cohesive brand identity.

### 2. Search Engine Optimization (SEO) & SEM Strategy
* Conducted deep-dive keyword research using **SEMrush** and **Google Keyword Planner** to establish a high-intent keyword mapping strategy for organic and paid growth.
